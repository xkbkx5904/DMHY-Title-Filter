# 动漫花园标题过滤器

一个为动漫花园增加标题过滤功能的油猴脚本，支持正则表达式、简繁体匹配和复杂过滤规则。应班友要求写的，感觉没什么用。

## 功能特点

- 实时过滤标题
- 支持简繁体自动匹配
- 支持正则表达式
- 支持复杂的过滤规则组合

## 过滤语法

1. **与运算**：使用分号（;或；）
   ```
   1080p；简体
   ```
   表示同时包含这两个条件

2. **或运算**：使用竖线（|）
   ```
   1080p|2160p；简体|繁体
   ```
   表示：(1080p或2160p) 且 (简体或繁体)

3. **正则表达式**：使用 /pattern/ 格式
   ```
   /\[第\d+集\]/；1080p
   ```
   表示：(标题包含[第n集]) 且 (包含1080p)

4. **简繁体匹配**：自动匹配简繁体变体
   ```
   简体
   ```
   会同时匹配：简体、簡體
